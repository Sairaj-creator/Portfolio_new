## 🚀 About This Project
This is an interactive, fully responsive 3D Glassmorphism portfolio built using **React** and **TypeScript**. 

### 🌟 Features
- ✨ **Dynamic 3D Glassmorphism** UI components
- 🌖 **Dark/Light Theme** with beautiful animated CSS Orbs
- ⚡ **Blazing Fast Performance** built with Vite
- 📱 **Fully Responsive** across all devices

### 💻 Quick Start
```bash
# Clone the repository
git clone https://github.com/Sairaj-creator/Portfolio.git

# Navigate into the project
cd Portfolio/frontend

# Install dependencies
npm install

# Start the dev server
npm run dev
```
